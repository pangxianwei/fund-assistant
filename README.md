
# 基金助手 Chrome 插件

简洁的基金净值查询工具

## 功能特点

- 输入基金代码快速添加基金
- 实时查看基金净值、估值和涨跌幅
- 数据每30秒自动刷新
- 支持添加多只基金同时监控
- 简洁美观的界面设计

## 安装方法

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹
6. 安装完成！

## 使用说明

1. 点击浏览器工具栏中的插件图标
2. 在输入框中输入6位基金代码（如：000001）
3. 点击"添加"按钮或按回车键
4. 查看基金的实时数据
5. 点击"删除"按钮可以移除不需要的基金

## 数据说明

- **净值**: 上一交易日的基金净值
- **估值**: 当前实时估算净值
- **涨跌幅**: 当日估算涨跌幅度
- **估值时间**: 数据更新时间

## 注意事项

- 基金代码必须是6位数字
- 数据来源于天天基金网，仅供参考
- 非交易时间数据可能不更新

## 常见基金代码示例

- 000001 - 华夏成长
- 110022 - 易方达消费行业
- 161725 - 招商中证白酒
- 320007 - 诺安成长混合

## 技术栈

- Chrome Extension Manifest V3
- Vanilla JavaScript
- Chrome Storage API
- 天天基金网API
