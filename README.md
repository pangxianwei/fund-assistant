# 基金助手 Chrome 插件

基金和大盘实时查看插件。

## 功能特点

- 输入 6 位基金代码快速添加基金
- 实时查看基金净值、估值、涨跌幅和估值时间
- 实时查看上证指数、创业板指行情
- 大盘和基金数据每 15 秒静默自动刷新一次
- 支持手动全局刷新
- 支持添加多只基金同时监控
- 紧凑弹窗布局，减少内部滚动

## 安装方法

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择本项目文件夹
6. 安装完成

## 使用说明

1. 点击浏览器工具栏中的插件图标
2. 在输入框中输入 6 位基金代码，例如 `000001`
3. 点击“添加”按钮或按回车键添加基金
4. 点击“刷新”按钮可以手动刷新全部大盘和基金数据
5. 查看顶部大盘区域中的上证指数、创业板指实时数据
6. 查看下方基金列表中的净值、估值、涨跌幅和估值时间
7. 点击“删除”按钮移除不需要的基金

## 数据说明

### 基金数据

- **净值**：上一交易日基金净值
- **估值**：当前实时估算净值
- **涨跌幅**：当日估算涨跌幅度
- **估值时间**：基金估值更新时间

### 大盘数据

- **指数**：当前展示上证指数、创业板指
- **最新点位**：指数最新成交点位
- **涨跌额**：相对前一交易时点的涨跌数值
- **涨跌幅**：指数实时涨跌幅
- **成交额**：指数对应市场成交额

## 刷新机制

- 首次打开弹窗时显示加载状态
- 后续自动刷新默认走静默更新，减少页面闪动
- 自动刷新间隔为 15 秒
- 手动点击“刷新”会立即更新全部数据

## 数据来源

- 基金估值数据：天天基金网接口 `https://fundgz.1234567.com.cn`
- 大盘指数数据：东方财富行情接口 `https://push2.eastmoney.com`

## 注意事项

- 基金代码必须是 6 位数字
- 数据仅供参考，请以交易平台实际数据为准
- 非交易时间部分数据可能不会变化
- Chrome 扩展弹窗本身存在高度限制，基金较多时仍可能出现滚动

## 常见基金代码示例

- `110022` - 易方达消费行业
- `161725` - 招商中证白酒
- `320007` - 诺安成长混合
